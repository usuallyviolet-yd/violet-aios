# Learning

This folder stores all learning-related notes, including the 30-Day AI Catch-up Plan, course notes, paper notes, tutorials, and daily learning logs.

## Main Rule

Learning notes go here when they are temporary, course-based, or part of my learning process.

Permanent concepts should later be summarized into `10_KNOWLEDGE/`.

## Structure

- `AI_Catchup/` — 30-day AI learning plan
- `Courses/` — course notes
- `Papers/` — paper reading notes
- `Tutorials/` — hands-on technical tutorials
